방법:
1. Window > Package Manager을 연다.
2. 좌측 상단에 + 버튼을 누르고 add package from git URL... 을 클릭한다.
3. https://github.com/mob-sakai/ParticleEffectForUGUI.git 주소를 넣어준다.
4. ClickEffect > VFX > Sample > Prefabs > ClickEffect 를 원하는 대로 바꿔준다.
